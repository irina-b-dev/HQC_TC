void hqc_pke_keygen ( unsigned char * pk,
unsigned char * sk )

Keygen of the HQC_PKE IND_CPA scheme.

The public key is composed of the syndrome s as well as the seed used to generate the vector h.

The secret key is composed of the seed used to generate vectors x and y. As a technicality, the public key is appended to the secret key in order to respect NIST API.

Parameters
[out]pkString containing the public key
[out]skString containing the secret key
