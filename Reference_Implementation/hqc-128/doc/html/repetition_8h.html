void repetition_code_decode ( uint8_t * m,
uint8_t * em )

Decoding the code words to a message using the repetition code.

We use a majority decoding. In fact we have that PARAM_N2 = 2 * PARAM_T + 1, thus, if the Hamming weight of the vector is greater than PARAM_T, the code word is decoded to 1 and 0 otherwise.

Parameters
[out]mPointer to an array that is the message
[in]emPointer to an array that is the code word

Encoding each bit in the message m using the repetition code.

For reasons of clarity and comprehensibility, we do the encoding by storing the encoded bits in a String (each bit in an unsigned char), then we parse the obtained string to an compact array using the function array_to_rep_codeword().
