void vect_fixed_weight ( uint8_t * v,
const uint16_t weight,
AES_XOF_struct * ctx )

Generates a vector of a given Hamming weight.

This function generates uniformly at random a binary vector of a Hamming weight equal to the parameter weight. The vector is stored by position. To generate the vector we have to sample uniformly at random values in the interval [0, PARAM_N -1]. Suppose the PARAM_N is equal to \( 70853 \), to select a position \( r\) the function works as follow:

  1. It makes a call to the seedexpander function to obtain a random number \( x\) in \( [0, 2^{24}[ \).
  2. Let \( t = \lfloor {2^{24} \over 70853} \rfloor \times 70853\)
  3. If \( x \geq t\), go to 1
  4. It return \( r = x \mod 70853\)

The parameter \( t \) is precomputed and it's denoted by UTILS_REJECTION_THRESHOLD (see the file parameters.h).

Parameters
[in]vPointer to an array
[in]weightInteger that is the Hamming weight
[in]ctxPointer to the context of the seed expander
